---
title: Web3Auth PnP Modal Angular Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Plug and Play Modal in your Angular app.

If you face any problem anytime, you can always find help in the [Web3Auth Community](https://web3auth.io/community/c/help-pnp/pnp-modal/7).

:::tip building locally

- Clone the PnP Modal Angular Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web-modal-sdk/quick-starts/angular-modal-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm run start
  ```

:::
