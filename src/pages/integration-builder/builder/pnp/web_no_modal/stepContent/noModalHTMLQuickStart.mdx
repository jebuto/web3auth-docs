---
title: Web3Auth PnP No Modal Vanilla JS Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Plug and Play No Modal in your Vanilla JS app.

If you face any problem anytime, you can always find help in the [Web3Auth Community](https://web3auth.io/community/c/help-pnp/pnp-no-modal/15).

:::tip building locally

- Clone the PnP No Modal Vanilla JS Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-pnp-examples/web-modal-sdk/quick-starts/vanillajs-no-modal-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm run start
  ```

:::
