---
title: Web3Auth PnP Unity Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Plug and Play in your Unity app.

If you face any problem anytime, you can always find help in the [Web3Auth Community](https://web3auth.io/community/c/help-pnp/pnp-unity/20).

:::tip building locally

- Clone the PnP Unity Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-pnp-examples/tree/main/unity/unity-quick-start w3a-quick-start
  ```

- Install & Run within your Unity Editor

:::
