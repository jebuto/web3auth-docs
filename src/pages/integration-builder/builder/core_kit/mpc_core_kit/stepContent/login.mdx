---
title: Logging in your User
---

Use the `connectTo` function in the Web3Auth Instance to login the user. Just create a button that triggers the `connectTo` for the user, for the
login method of their choice.

After a successful user login, the `connectTo` the function returns a provider which can be used to interact with the blockchain and sign
transactions.

- [`connectTo()` function](/sdk/pnp/web/no-modal/usage#logging-in-the-user).

:::
