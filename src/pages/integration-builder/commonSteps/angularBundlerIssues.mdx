---
title: Fixing Bundler Issues
---

While using Web3Auth in Angular, you may run into issues building. This issue occurs because some core packages like `eccrypto` have certain
dependencies which are not present within the browser build environment.

:::tip Troubleshooting

To solve this, please have a look at our troubleshooting pages:

- [Webpack 5 Issues](/troubleshooting/webpack-issues)
- [Vite Issues](/troubleshooting/vite-issues)

:::
